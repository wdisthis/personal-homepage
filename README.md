# Personal Dashboard

A high-density, technical analytic dashboard for quick access to tools, links, and projects.

## Features
- **Analytic UI**: Dense, industrial-style dashboard.
- **Google Search**: Integrated terminal-style search bar.
- **Live Clock**: Real-time clock and date display in English.
- **Link Filtering**: Dynamic filtering of cards as you type.
- **Responsive**: Works on desktop and mobile.

## Project Structure
- `index.html`: Main entry point.
- `style/`: Contains CSS styles.
- `script/`: Contains JavaScript logic.
- `assets/`: Contains images and other media.
- `outline.txt`: Source outline for categories.
